Advanced Table Formatting

HTML tables start with a table tag. Add class="table" to make it a Bootstrap table. Add class="table table-striped" to make it a striped table.

Table rows start with a tr tag.
Table data start with a td tag.
Add class="active" to any table row or cell. Other valid classes include info, warning, active, danger, and success.
